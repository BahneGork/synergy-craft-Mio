How to Use

Browsing Items

  • Use the search bar to filter by item name or class name
  • Use the Class filter to show only gear for a specific class
  • Ingredient names are colour-coded: orange = rare drops, purple = farmable materials, blue = Stronghold
  • Click any ingredient name to see which items need it (shown in the right panel)

Shopping List

  • Click + Add to List on a card to add it to your shopping list
  • The left panel shows combined ingredient totals across all selected items
  • Selected items float to the top of the grid for easy access
  • Export your list with the TXT / MD / CSV buttons at the bottom of the panel

Progress Tracking

  • Track gathering in the Gathering ledger by location — enter how many of each ingredient you have
  • When gathered ≥ needed, ingredient rows turn green across all views
  • Progress is saved automatically and persists across browser sessions and shutdowns
  • Use Reset Gathered to clear progress while keeping your shopping list
  • Use Reset All Progress to wipe everything and start fresh

Gathering ledger by location

  • Click ☷ Gathering ledger by location to switch from item cards to a flat table
  • Select a location (e.g. a dungeon or explorer chart) from the dropdown
  • The table shows every ingredient from that location — update counters while you farm
  • Gathered amounts are shared across all items that need the same ingredient

Ingredient Lookup

  • Click any ingredient name on a card to populate the Used In panel on the right
  • Shows all crafting items that require that material and the quantity needed for each
